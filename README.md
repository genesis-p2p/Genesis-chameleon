# Project Chameleon — Phase 1 (MVP)

This is the minimal Proof-of-Concept release for the Chameleon covert-channel project.

## What Works in Phase 1

- Hide data inside PNG using LSB steganography
- Extract hidden data from image
- FastAPI endpoints:
  - POST /hide
  - POST /extract
- Fully modular engine structure
- Unit test included
- Clean architecture prepared for future phases

This release demonstrates the core idea with a functional prototype.
